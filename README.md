### pixi.js 初体验

其中包含了基本使用的`example`代码，位于`pages`文件内

本项目使用了`React hooks`，如不了解请先参阅`React`官方文档

#### 安装
```
yarn
```
or
```
npm i
```

### 启动
```
npm start
```
or
```
yarn start
```
